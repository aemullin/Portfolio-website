Movielist API

Movielist vAPI, is a RESTful API that communicates with a mongoDB database containing information on movies and users. This API allows users to register and view information on movies and their directors in addition to allowing the user to add and remove movies to a favorites list. This API combines Node.js, MongoDB and Mongoose.
View this project on github!
